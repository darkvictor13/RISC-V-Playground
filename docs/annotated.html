Simulador de máquina RISC-V
Lista de Classes
Aqui estão as classes, estruturas, uniões e interfaces e suas respectivas descrições:
 CAddA classe Add: Classe base para todos os subcircuitos que realizam a soma de 2 valores
 CAddBranchA classe AddBranch: Circuito somador que soma ao program counter o offset
 CAddNextA classe AddNext: Calcula para onde o Pc deve apontar para obter a próxima instrução
 CALUA classe ALU: Representa a unidade lógica e aritimética responsável por executar toda operação do programa
 CALUControlA classe ALUControl: Responsável por indicar que operação a ALU deve executar
 CAssemblerA classe Assembler: responsável por gerar o código binario a partir das instruções assembler para os processadorees da arquitetura RISC-V
 CControlA classe Control: Ativa as flags específicas para cada instrução
 CDataMemoryA classe DataMemory: Representa a memória de Dados
 CImmGenA classe ImmGen: Gera o imediato para as instruções
 CInstructionA classe Instruction: é a base para todos os modos de instrução
 CInstructionMemoryA classe InstructionMemory: Representa a memória de instruções
 CInstructionTypeIA classe InstructionTypeI: representa uma instrução do tipo I
 CInstructionTypeRA classe InstructionTypeR: representa uma instrução do tipo R
 CInstructionTypeSA classe InstructionTypeS: representa uma instrução do tipo S
 CInstructionTypeSBA classe InstructionTypeSB: representa uma instrução do tipo SB
 CLabelA estrutura Label: responsável por representar um label dentro do código assemble, para isso armazena a o nome do label e a posição em que este se encontra
 CMakeBranchA classe MakeBranch: Verifica se há condições de realizar o breanch
 CMemoryA classe Memory: Classe abstrata base para os diferentes tipos de memória
 CMuxA classe Mux: Classe abstrata base para os diferentes tipos de multiplexador
 CMuxTypeAA classe MuxTypeA: Multiplexador para selecionar o valor que será colocado do program counter
 CMuxTypeBA classe MuxTypeB: Multiplexador para selecionar o valor que será colocado na segunda entrada da ALU
 CMuxTypeCA classe MuxTypeC: Multiplexador para selecionar o valor que será colocado no Registrador
 CPCA classe PC: Indica qual a instrução atual
 CRegistersA classe Registers: Representa os registradores visíveis ao programador
 CSimulatorA classe Simulator: Agrupa todos os componentes necessários para a simulação
 CViewMainWindowA classe ViewMainWindow: responsável pela representação dos dados de estado do simulador en tela, alem de permitir a interação do usuário com o simulador e o assembler
 CWordA classe Word: Representação de uma palavra de 32 bits e suas operações
